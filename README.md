# Recipe Index

This project is a full-text searchable index of the [RecipeNLG dataset](https://recipenlg.cs.put.poznan.pl) of over 2 million recipes.

## Technology

- Vue
  - Frontend framework
- Nuxt
  - Server-side rendering
- DaisyUI
  - Styled components
- TailwindCSS
  - CSS framework
- Meilisearch
  - Full-text search engine
